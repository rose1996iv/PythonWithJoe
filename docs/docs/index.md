## üõ†Ô∏è Getting Started Locally

Want to run this documentation on your local machine? Follow these simple steps:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Joseph1997-eng/Let-Us-Python-Book-.git
    cd Let-Us-Python-Book-
    ```

2.  **Install dependencies**
    ```bash
    pip install mkdocs-material
    ```

3.  **Run the server**
    ```bash
    mkdocs serve
    ```

4.  **View in Browser**
    Open [http://127.0.0.1:8000](http://127.0.0.1:8000) to start reading.
