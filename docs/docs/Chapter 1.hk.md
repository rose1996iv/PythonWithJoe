## 1.1 Python Timi Cu Zeidah A Si?

- Python cu **high-level programming language** a si i, **Guido van Rossum** nih a sermi a si. Amah cu capo men in Python i *Benevolent Dictator For Life (BDFL)* tiah an auh.
- **1991 Kum** ah a voikhatnak an chuah.
- Python interpreters pawl cu Operating Systems tampi **Windows**, **Linux**, le a dangdang pawl ah hman khawh a si.
- Python programmer pawl kha **Pythonists** asiloah **Pythonistas** tiah an auh hna.

---

## 1.2 Python A Lar Tuknak A Ruang Pawl

Python hi a lar tuknak a ruang tampi a um:

### (a) A Man Pek A Hau Lo (Free)

- Python cu **a man pek hau lo in hman le ṭhenh (distribute)** khawh a si.
- **Open-source community** nganpi nih an bawmh/support.
- Python interpreter cu **platform (OS) ngan pawl dihlak** ah hmuh khawh a si.

### (b) Software A Ṭhat Ning (Software Quality)

- Python code a ṭhat ning hi **traditional scripting languages** dang nak in a ṭha deuh tawn.
- Python nih **code rel fawi** kha a biapi in a chiah, cucaah hman ṭhan (reuse) le zohkhenh (maintain) a fawi.
- Amah nih **advanced reuse mechanisms** a simi modules, packages, le object-oriented programming te hna a support.

### (c) Developer Pawl Rian Ṭuan A Raan (Developer Productivity)

- Python cu statically typed languages tampi nak in **rian a ṭuan a rang (productivity a sāng)**.
- Programs pawl an **tawi** deuh i an **fawi** deuh.
- **Type, debug, le maintain** tuah ding a tlawm deuh.
- A caan a la tukmi **compile le link** steps pawl a hau lo.

### (d) Program A Dang Ah Ṭhial A Fawi (Program Portability)

- Python programs cu platforms tam deuh ah thlen hau lo in **unchanged** (a ning tein) a run kho.
- Python cu atu lio hman mi **platforms ngan pawl dihlak** ah a run kho.
- Platform thar ah program ṭhial a hauh tik zong ah thil tlawmpal te lawng (copy–paste) thlen a hau tawn.

### (e) Bawmtu Libraries Pawl (Support Libraries)

- Python nih **standard library support** ṭhawng taktak a ngei.
- Third-party libraries pawl tampi um mi.

### (f) Hmun Khat Ah Fonh Khawhnak (Component Integration)

- Python nih **C le C++ libraries** le **Java components** pawl a au/hmang kho.
- **COM** le **.NET** frameworks pawl he zong rian a ṭuan kho.
- Network cung ah **SOAP**, **XML-RPC**, le **CORBA** hman in rian a ṭuan kho.
- Glue code ṭha tein hman ahcun, Python nih **C++, Java, le C# classes pawl a subclass** kho.

### (g) Nuamhnak (Enjoyment)

- Python cu **cawn a fawi i hman a fawi**.
- Built-in toolset ṭha tampi a um.
- Programming cu rian ṭuan nak in nuamnak a si deuh.

## 1.3 Python Hi Zei Nih Dah A Dang (Special) Siter?

### (a) A Ṭhawng (Powerful)

- **Dynamic typing**
- **No variable declaration**
- **Automatic allocation le garbage collection**
- Modules, classes, exceptions support
- Powerful containers — Lists, Dictionaries, Tuples, etc.

### (b) Tuah Cia Mi Thil Pawl (Ready-made Stuff)

- Join, slice, sort, map le operations hmang thei.
- Library tampi um.

### (c) Hman A Fawi (Ease of Use)

- **Type le run**
- Compile le link steps a om lo
- Interactive programming
- Program cu tawi, hme, flexible

## 1.6 Programming Paradigms (Rian Ṭuan Ning)

### 1.6.1 Programming Paradigm Timi Cu Zeidah A Si?

- Paradigm = organizing principle/model  
- Programming paradigm = program tuahnak style  
- Functional, Procedural, Object-oriented, Event-driven  
- Python nih pakhat nak tam support

---

### 1.6.2 Functional Programming Model

- Problem cu **functions** ah a ṭhen.
- Input → Output.
- Python FP tools:
  - lambda  
  - map()  
  - reduce()  
  - filter()

- Tahchunhnak: Factorial & Fibonacci

```text
factorial(n) = 1 if n == 0
= n * factorial(n - 1)

fibo(n) = 0 if n == 0
= 1 if n == 1
= fibo(n - 2) + fibo(n - 1)
```

---

#### Functional Programming – Properties

- Output cu input ah hngat.
- Same input → same output.
- Side effects a um lo.
- Parallel execution a ṭha.
- Declarative style.

---

### 1.6.3 Procedural Programming Model

- Statements cu order nei in tuah.
- Functions cu math functions a si lo.
- State a thlen thei.

#### Procedural Programming – Properties

- Statements cu value a pek lo — state a thlen.
- State dang ih an run tik ah result khal dang thei.
- Global state a thlen.
- Imperative.

---

### 1.6.4 Object-oriented Programming (OOP) Model

- Real world thil pawl cu object in model.
- Example objects:
  - VC  
  - Professors  
  - Students  
  - Courses  
  - Exams  

#### Object Concepts

- State → attributes  
- Behavior → methods  
- Class in suahmi  
- Method call in messages an peh

---

### 1.6.5 Event-driven Programming Model

- GUI ah a hman tuk.

#### Zeitin a tuan?

- User click/scroll → event a cang.
- Listener functions → react.

#### Event-driven Programming – Properties

- Order a fixed lo.  
- User interaction ah hngat.  
- Asynchronous.  
- Hmanmi:
  - GUI  
  - Web front-end  
  - Games
